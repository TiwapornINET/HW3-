ข้อมูลหลักสูตร

ชื่อหลักสูตร : หลักสูตรวิศวกรรมศาสตรบัณฑิต สาขาวิชาวิศวกรรมสารสนเทศและเครือข่าย

รหัสหลักสูตร : 67062043

ระดับการศึกษา : ปริญญาตรีเทียบโอน 2-3 ปี

คณะ : คณะเทคโนโลยีและการจัดการอุตสาหกรรม

มหาวิทยาลัย : มหาวิทยาลัยเทคโนโลยีพระจอมเกล้าพระนครเหนือ วิทยาเขตปราจีนบุรี

หลักสูตรวิศวกรรมศาสตรบัณฑิต(วศ.บ.)(เทียบโอนปวส.) สาขาวิชาวิศวกรรมสารสนเทศและเครือข่าย ระยะเวลาการศึกษา 3 ปี เป็นหลักสูตรเสริมภาษาอังกฤษ
